Welcome to Pathfinding Visualizer!

This tool helps you visualize how different pathfinding algorithms work on a grid.

Getting Started:

  • First click places a start point (orange)
  • Second click places an end point (turquoise)
  • Additional clicks create walls/obstacles
  • Right-click to remove a node
  • Press Space or click "Visualize" to run the algorithm

Special Features:

  • Toggle between 2D and 3D visualization
  • Use mouse wheel to zoom in/out
  • Hold Shift + drag to pan around (when zoomed in)
